# HOHLOMA — Тату, барбершоп, пирсинг, массаж в Пензе

Современный сайт для студии «Хохлома» с удобной системой онлайн-записи.

---

## О проекте

Сайт разработан для студии «Хохлома», которая объединяет четыре направления: татуировки, барбершоп, пирсинг и массаж. Главная задача — дать клиентам простой и понятный инструмент для записи к мастерам, а администраторам — удобный способ собирать заявки.

**Сайт доступен по ссылке:**  
[https://cheech1ko.github.io/HOHLOMA/](https://cheech1ko.github.io/HOHLOMA/)

---

## Как это работает

### Для клиента

1. Выбирает раздел (тату, барбершоп, пирсинг, массаж)
2. Смотрит мастеров, портфолио, цены
3. Нажимает «Записаться»
4. В модальном окне:
   - выбирает услугу
   - выбирает мастера
   - выбирает дату и время
   - оставляет имя и телефон
5. Получает уведомление, что запись сохранена

Если клиент нажимает «Записаться» из карточки мастера — форма сразу подставляет выбранного мастера.

### Для администратора

Сейчас все записи сохраняются в Google Таблицы, где администратор сможет:

- видеть все записи в одном месте
- отслеживать загруженность мастеров
- вести учёт клиентов

## Структура сайта

- **Татуировки** — мастера, портфолио(будет добавлено позже), цены (индивидуальные эскизы, перекрытия)
- **Барбершоп** — стрижки, борода, бритьё, прайс с разбивкой по уровню мастеров
- **Пирсинг** — мастера, украшения, таблица цен на проколы
- **Массаж** — виды массажа, цены, рекомендации

---

## Технологии

- HTML5, CSS3 (Flexbox, Grid, адаптивная вёрстка)
- JavaScript (чистый, без jQuery)
- GSAP — анимации при скролле
- Flatpickr — календарь для выбора даты
